# ML-DSA Tool

This script usees the Python `cryptography` library to access `openssl` for `mldsa` (ML-DSA).

The ml_dsa.py tool is run on a system to:

- generate ML-DSA (Dilithium) private keys
- private key files are encrypted PEM PKCS8 with interactive passphrase input
- sign files with ML-DSA private keys
- verify ML-DSA signatures for files
